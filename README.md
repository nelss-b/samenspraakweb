# Samenspraak Hackathon Website

Static one-page presentation site for Samenspraak.

## Update the demo video

Open `assets/script.js` and replace:

```js
const demoVideoUrl = "";
```

with a YouTube, Vimeo, or Loom share URL.

## Publish with GitHub Pages

1. Push this repository to GitHub.
2. Open the repository settings.
3. Go to Pages.
4. Set source to `Deploy from a branch`.
5. Choose branch `main` and folder `/(root)`.
6. Save. GitHub will show the public URL after the first deploy.
